공지사항

안녕하세요 위상세팅 시뮬레이션 제작자입니다.
1. 각 직업별 URL이 생성되었습니다.
상단 직업 버튼을 누르면 해당하는 직업 URL로 변경됩니다.
직업 URL로 저장해두면 페이지 접근시 초기 원하는 직업으로 볼 수 있습니다. 2. 장비 위상/보석 셋팅 URL 기능이 추가 되었습니다.
위상/보석을 셋팅할때마다 주소창의 URL이 선택한 값들을 저장합니다.
주소창의 URL copy하고 paste하면 세팅된 장비목록을 로드합니다. (현재 위상/보석 셋팅만 가능) 3. 위상 획득처가 표기됩니다.
위상 선택하면 해당 위상의 획득처가 표기됩니다.
전설 아이템에서만 얻을 수 있는 위상은 위상 추출로 표기됩니다.
고유 아이템의 경우 월드 드랍으로 표기됩니다.
